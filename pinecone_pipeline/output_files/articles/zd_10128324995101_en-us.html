Ledger Recover One-Time Security Code

Your One-Time Security Code provides an additional level of security to Ledger Recover. The code will be displayed on the Ledger device that you're using to restore access to your wallet. You'll be asked to provide the code during the identity verification process. If you can't provide the code, you'll need to start over. Each One-Time Security Code can only be used once. If you start over, you'll get a different code.

Here are some tips to help you:

  • Note down the code and keep it close to you.
  • Allow access to your camera and microphone.
  • Choose the language in which you feel most comfortable saying the code.
  • Say one number at a time in a quiet place with a stable internet connection.
  • Do not share or use the code in any other way. Only use a code that is displayed on your Ledger device.
