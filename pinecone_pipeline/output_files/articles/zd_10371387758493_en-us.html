Metamask x Ledger Connection Issue On Firefox

Firefox version 114.0 and above no longer support connecting Ledger devices to Metamask, resulting in an error.

This change is due to Firefox dropping support for the legacy U2F JavaScript API.

Workaround

  • To continue using Metamask with your Ledger, we recommend switching to Chrome or Brave until a fix is deployed by the Firefox team.
  • Alternatively, you have the option to install an older version of Firefox; however, we do not recommend this for security reasons.
