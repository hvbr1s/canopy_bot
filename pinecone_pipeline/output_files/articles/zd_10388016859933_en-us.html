Compare Ledger Live Mobile and Desktop

This article compares the main differences between Ledger Live Desktop and Ledger Live Mobile. 

Feature Ledger Live Desktop Ledger Live Mobile
Export CSV Yes No
Export accounts from Ledger Live Desktop Yes No
Import accounts from Ledger Live Desktop No Yes
Update firmware Yes Yes¹
Send crypto Yes Yes²
Receive crypto Yes Yes
Send NFT Yes³ Yes³
Stake crypto Yes Yes
Pairing via Bluetooth No Yes⁴
Apps in the Discover section Yes Yes⁵
  1. Firmware on Ledger Live Mobile can be updated only on Android devices via a USB-connected Ledger device.
  2. Sending for Nano S and Nano S Plus is available only on Android devices with OTG standard via a USB-connected Ledger device.
  3. Ledger Nano S does not support send NFT feature on both Desktop and Mobile. Sending NFTs with Nano S Plus on Mobile is available only on Android devices via a USB-connected Ledger device.
  4. Only available for Ledger Nano X.
  5. Discover section on iOS devices does not include apps.
