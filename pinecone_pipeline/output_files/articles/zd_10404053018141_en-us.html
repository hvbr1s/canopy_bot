API HTTP 503 error on Solana account

This article explains how to solve API HTTP 503 errors on Solana accounts.

API HTTP 503 error

The error API HTTP error may appear on adding a Solana account in Ledger Live. It also can result in a synchronization error or in an error on a staking transaction.

This error may appear from time to time when there are too many requests at the moment to our infrastructure. 

Troubleshooting

First of all, make sure to have the latest setup of your Ledger Live, Ledger device, and the apps on the device.

Find detailed instructions on updates below:

  • update Ledger Live 
  • update Nano S / Nano X / Nano S Plus
  • update all apps on the Ledger device

If the issue persists, in some cases, retrying the transaction after some time can solve the problem.

Alternative solution

Use Ledger Live Mobile

Using Ledger Live Mobile might be a solution while the issue persists on Ledger Live Desktop.

Use third-party wallet

While our team is working on improving the situation, please use a Ledger device with third-party wallets to manage your Ledger Solana accounts: 

  • Connecting your Ledger Solana (SOL) account to Solflare
  • Connecting your Ledger Solana (SOL) account to Phantom wallet
