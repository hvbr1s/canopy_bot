Expanse (EXP)

Install the Expanse app on your Ledger device to manage EXP tokens with MetaMask or MyCrypto. The Expanse app is developed and supported by Ledger. Check the Expanse wallet page to learn more.

Before you start

  • Set up Ledger Live with your Ledger device.
  • Update the firmware of your Ledger Nano S / Nano S Plus / Nano X.
  • Google Chrome or Firefox is installed.
  • Install the Ethereum app, as Expanse requires it.

Install the Expanse app

  1. Open My Ledger in Ledger Live.
  2. Connect and unlock your device.
  3. If asked, allow My Ledger to access your device.
  4. Find Expanse in the app catalog.
  5. Click the Install button of the app.
    • An installation window appears.
    • Your device will display Processing... 
    • The app installation is confirmed. 

Manage EXP on MetaMask

Add the Expanse Network to MetaMask

MetaMask is a software cryptocurrency wallet used to interact with the Ethereum blockchain and all EVM chains (BNB Chain, AVAX, Polygon, etc.). It allows users to access their wallets through a browser extension or mobile app, which can then be used to interact with decentralized applications.

  1. Download and install the MetaMask extension in your browser by following these steps.
    We strongly recommend using Chrome or Firefox browsers.
  2. Click on the top-right menu of the MetaMask wallet.
  3. Select Settings.
  4. Select Networks.
  5. Click Add Network.

Use the RPC data below to connect MetaMask to Expanse:

Network Name: Expanse
New RPC URL: https://node.expanse.tech
Chain ID: 2
Currency Symbol: EXP
Block Explorer URL: https://explorer.expanse.tech

Thats it! You've successfully added the Expanse network to your MetaMask account. 

Send/Receive tokens

To send or receive a transaction, always verify all transaction details on your device.

  • To receive, verify the receive address on your device.
  • To send, verify the amount, recipient address, and gas rate. Approve on your device to sign and broadcast the transaction.
