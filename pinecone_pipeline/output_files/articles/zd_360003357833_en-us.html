Change your PIN code

The PIN code of your Ledger Nano S device prevents unauthorized access to your crypto assets. Your PIN code is chosen when you first set up the device, but you can change it at any time. 

Instructions

  1. From the dashboard, navigate to Settings > Security > Change PIN.
  2. Choose a new PIN code of 4 to 8 digits.
  3. Confirm the new PIN code by entering it again.
  4. Enter your old PIN code to validate.
  5. Your new PIN code is now set.

  Security tips

  • Choose your own PIN code. This code unlocks your device.
  • An 8-digit PIN code offers an optimum level of security.
  • Choose a PIN code that's hard to guess.
