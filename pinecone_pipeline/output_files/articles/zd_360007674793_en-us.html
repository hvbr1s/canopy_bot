Wanchain (WAN)

Install the Wanchain app on your Ledger device to manage WAN with the WanWallet Desktop. The Wanchain app is developed and supported by the Wanchain community. Check the Wanchain wallet page to learn more.

  Wanchain accounts cannot be added to Ledger Live.

Before you start

  • Set up Ledger Live with your Ledger device.
  • Update the firmware of your Ledger Nano S / Nano S Plus / Nano X.
  • Install the Ethereum app, since Wanchain requires it.

Install the Wanchain app

  1. Open My Ledger in Ledger Live.
  2. Connect and unlock your Ledger device.
  3. If asked, allow My Ledger to access your device.
  4. Find Wanchain in the app catalog.
  5. Click the Install button of the app.
    • An installation window appears.
    • Your device will display Processing... 
    • The app installation is confirmed.

  If you get an error message that the Bitcoin app is required, please install the Ethereum app. The accuracy of the error message will be improved.
