Changes to the DAI stablecoin

Maker has announced that it will rename its stablecoin Dai, an ERC20 token, to Sai in order to distinguish between single-collateral and multi-collateral Dai. Original Dai (Sai) users can migrate to the new Dai token (Dai) using this migration tool.

Changes in Ledger Live

In Ledger Live version 1.19 the following changes take effect:

  • The ERC20 token previously known as Dai, will be listed as Dai Stablecoin v1.0 (SAI).
    The contract address remains 0x89d24a6b4ccb1b6faa2625fe562bdd9a23260359. Dai holders will find their coins will be listed as SAI.
  • A new ERC20 token will be listed as Dai Stablecoin v2.0 (DAI).
    The contract address is 0x6b175474e89094c44da98b954eedeac495271d0f. Original Dai (SAI) holders can migrate to the new Dai using Maker's migration tool.
