IOTA (MIOTA)

  Chrysalis network upgrade 

Following the Chrysalis upgrade on April 28th, 2021, the Trinity wallet was replaced by the Firefly wallet. If you managed IOTA with your Ledger device before the network upgrade, you need to migrate your tokens to the new Chrysalis network before you can start using the Firefly wallet. You can follow the Chrysalis Migration Guide to help you through the migration process.

Install the IOTA app on your Ledger hardware wallet to manage MIOTA with the official Firefly wallet. The IOTA app is developed and supported by the IOTA community. Check the IOTA wallet page to learn more.

 IOTA accounts can not be added to Ledger Live.

Before you start

  • Set up Ledger Live with your device.
  • Update the firmware of your Ledger Nano S / Nano S Plus / Nano X.

Install the IOTA app

  1. Open My Ledger in Ledger Live.
  2. Connect and unlock your Ledger device.
  3. If asked, allow My Ledger to access your device.
  4. Find IOTA in the app catalog.
  5. Click the Install button of the app.
    • An installation window appears.
    • Your device will display Processing...
    • The app installation is confirmed.

Next steps

  • Check the Firefly user guide for setup instructions.
  • Reach out to the IOTA community on Discord if you need assistance.
