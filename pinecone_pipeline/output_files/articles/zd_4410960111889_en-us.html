Update Ledger Live

This tutorial describes how to update Ledger Live Desktop on your PC.

  Important notice 

  • Users on version 2.29 and below won’t be able to perform auto-update in Ledger Live due to different signature certificates used in old versions. As a result, they see a stuck “Downloading update” bar in Ledger Live.
  • The only solution will be to update Ledger Live manually by downloading the Ledger Live executable file from the website.
 Attention MacOS Users

If you're having issues installing Ledger Live after downloading it within the app, please make sure you’ve moved Ledger Live to the applications directory before installing it.

Before you start

  • Make sure to have a computer with at least macOS 10.15, Windows 8.1 (64-bit), Linux Ubuntu 20.04 (64-bit). Learn more
  • You have a stable internet connection.
Auto-update in Ledger Live Manual update of Ledger Live

Once a new version of Ledger Live is released, it will progressively become available for all Ledger users.

  1. If you see a banner with the available update in Ledger Live, click Download update.
    The new version downloads and then the button Install now appears.
  2. Click Install now.
    Ledger Live closes and the installation starts.
  3. Once the installation is complete, click Finish.
    A new version of Ledger Live will launch.

    Your Ledger Live is updated to the latest version.

In rare cases, the manual update of Ledger Live is needed. If your Ledger Live is outdated and you are not able to auto-update in Ledger Live, you need to download the latest installation file from the website. You do not need to uninstall your current version of Ledger Live.

After the update, all the settings and accounts will remain in the new version.

  1. Go to the Ledger website.
  2. Download the Ledger Live installation file for your operating system.

  3. Once the installation file is downloaded, install Ledger Live.
    Your Ledger Live is now running the latest version.

How do I update Ledger Live Mobile?

  • Once the new update is available on the OS of your mobile phone (iOS/Android), you should see the update in App Store and Google Play.
  • The new versions of Ledger Live might be released on different days for Mobile and Desktop.
