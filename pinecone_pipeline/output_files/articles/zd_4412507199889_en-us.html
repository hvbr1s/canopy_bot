I can't find an app in My Ledger

Are you trying to install an app to your Ledger device but you can’t find it in the list?

The article will explain why it is happening and how you can solve it.

The asset has no app because it is a Token

If you are looking for a dedicated app for assets like USDC, BTT, DAI, or others, and you can't find it in the App catalog in My Ledger, this is perfectly normal. There is no dedicated app because you are trying to manage an ERC20 / BEP20 / TRC-10/20 token.

An app allows your Ledger device to manage different assets on a specific network. For instance, with the Ethereum (ETH) app, you can sign transactions and manage different assets (ERC20, ERC721...) on the Ethereum network, the same goes for BSC.

This is why ERC20 tokens (or other standardized tokens like BEP20 or TRC10/20) do not need a dedicated app. You only need the app of the network you wish to use:

Token App required Tutorial
ERC-20 Ethereum (ETH) Manage ERC-20
BEP-20 Binance Smart Chain (BNB) Manage BEP-20
TRC-10/20 Tron (TRX) Manage TRC-10/20
Crypto is not supported by Ledger hardware wallets
Sometimes you can’t find an app in the app catalog in Ledger Live because no app has been developed yet for the network you wish to use. 

Ledger hardware wallets benefit from a secure operating system called BOLOS, providing an open platform upon which developers can build apps. Developers willing to build apps for Ledger hardware wallets, like adding support for the crypto asset you're referring to, are welcome to get in touch with us.

You can check our list of supported crypto assets to see which assets are supported on the Ledger device. Additionally, if you are curious about the coming assets, a public project overview is available here.
Outdated Ledger Live and/or outdated firmware

You might not be able to install the latest app because you are running outdated Ledger Live or an outdated firmware of your Ledger device. 

To avoid that, make sure to have the latest version of Ledger Live, you can always find the information about the latest version here.

  • If your Ledger Live is outdated, you can update it automatically or manually by following the instructions from our Help Center

Secondly, make sure you are running the latest firmware version on your Ledger device. You can find information about the latest firmware releases for Nano S and for Nano X.

  • If your Ledger device is outdated, you can update your firmware by following the instructions for Nano S and Nano X.
