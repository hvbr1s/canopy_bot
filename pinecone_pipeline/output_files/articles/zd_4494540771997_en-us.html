Ledger Nano S Plus Firmware Release Notes

Release notes for all Ledger Nano S Plus firmware versions are chronologically listed in this article. Please check our step-by-step update guide here.

Firmware version 1.1.0 (latest version)

Released 14 February 2023

What’s new in OS version 1.1.0?

The OS version 1.1.0 features support for French and Spanish. Keep in mind that you can only change the language of the Ledger Nano S Plus operating system. The apps are available in English.

Before you update

  • Make sure you have updated Ledger Live through the notification banner or downloaded the latest version of Ledger Live. 

Enhanced user experience

  • In addition to English, you can now set the device's language to Spanish or French. To change the language, navigate to My Ledger → Language in the Ledger Live app.

Note: Changing the language of your Ledger Nano S Plus doesn't automatically change the language of Ledger Live.

Firmware version 1.0.4

Released 27th September 2022

What’s new in firmware version 1.0.4?

Firmware version 1.0.4 brings a redesigned PIN code screen for Ledger Nano S Plus. 

Before you update

  • Make sure you have updated Ledger Live through the notification banner or downloaded the latest version of Ledger Live. 

Better user experience

  • Redesigned the PIN code screen to improve security and facilitate PIN code entry on your Ledger Nano S Plus.

Fixes 

  • Fixed various bugs and improved stability.
