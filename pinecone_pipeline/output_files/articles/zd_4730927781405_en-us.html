How to add Ledger Live Desktop to the whitelist of Windows Security

This article explains how to add Ledger Live Desktop to the exclusions list of Windows Security on Windows 10.

It can be helpful in troubleshooting the connection issues of your Ledger device with a PC or in some cases when Microsoft Defender blocks Ledger Live from launching. Learn more

Add Ledger Live Desktop to the whitelist of MS Defender

  1. Click on Start  .
  2. Type Settings   and open it.
  3. Choose Update & Security  .
  4. Open Windows Security.
  5. Find Virus & threat protection and under it, select Manage settings.
  6. Scroll down and find Exclusions.
  7. Select Add or remove exclusions.
  8. Click on Add an exclusion.
  9. Choose Folder.
  10. Select Ledger Live folder (directory by default C:\Program Files\Ledger Live)
