Solving 0x6a82 error

This article explains how to solve an 0x6a82 error that you may face in Ledger Live when adding a BTC account.

Solution

In order to solve this error, please check if you disabled all experimental features in Settings in Ledger Live.

In particular, check if the Extended Account Search feature is disabled:

  1. Go to Settings in Ledger Live.
  2. Select the Experimental features tab.
  3. Turn off Extended Account Search if it is enabled.
  4. Restart Ledger Live by closing it and opening it again.
  5. Try again to add BTC account.
