Can I run Ledger Live on a Chromebook?

No, Ledger Live is not supported on Chromebook computers.

Chromebook computers are designed to run web-based apps whereas Ledger Live is a local app that needs to be installed on your computer's hard drive. Chromebooks don't have a substantial hard drive and instead rely on cloud storage to store data.

Instead, we recommend installing Ledger Live on a Windows, macOS or Linux machine that meets the minimum configuration requirements.
