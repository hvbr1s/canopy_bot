Can I run Ledger Live on an iPad or tablet?

We do not recommend running Ledger Live on an iPad or tablet.

While you might be able to install the app, Ledger Live wasn't designed to run on tablets. As such you might run into multiple display and software bugs that might considerably degrade your experience of using Ledger Live.
