Ledger Extension FAQ

This article answers the most common questions about Ledger Extension.

Rather than continue updating Ledger Extension as a standalone app, we’re integrating many of its features such as Web3 checks into other parts of the Ledger ecosystem to streamline your experience and make it easier to securely manage your assets.

You can still use the Ledger Extension app if you have it already, but it will no longer be available to download, and we have no plans for further updates.

Ledger Extension basics

What is Ledger Extension?

Ledger Extension is a multi-chain browser extension that allows you to connect directly to NFT marketplaces and dApps using your Ledger Nano X and warns you about malicious and suspicious websites and smart contracts before signing transactions.

Do I need a Ledger wallet to use Ledger Extension?

Using a hardware wallet is the most secure way to interact with Web3. Therefore, you need to use a Ledger Nano X with Ledger Extension.

Do I need the Ledger Live app to use Ledger Extension?

You can use Ledger Extension without the Ledger Live app. If you want to buy, sell or swap crypto, you'll need to connect Ledger Extension to a dApp that provides such services, for example, Uniswap. If you want to use the buy, send, receive and swap features of the Ledger ecosystem, the Ledger Live app is required.

Compatibility with browsers, dApps, and chains

With what browsers is Ledger Extension compatible?

Ledger Extension is compatible with Safari on iOS 16+ and macOS 12+ with Safari 16+.

Are all Ledger wallets compatible with Ledger Extension?

Ledger Nano X is compatible with Ledger Extension thanks to its Bluetooth feature.

What dApps are compatible with Ledger Extension?

You can connect to the following dApps using Ledger Extension:

  • OpenSea
  • PancakeSwap
  • Curve
  • Zapper
  • Manifold
  • Revoke.cash

To connect, you need to click the Ledger wallet button, which is your gateway to the Ledger ecosystem. If your device is compatible, you'll be prompted to install Ledger Extension.

You can use the MetaMask button if the Ledger wallet button isn't available.

What chains does Ledger Extension support?
Ledger Extension supports Ethereum and Polygon.

Ledger Extension vs other browser extensions

What's the difference between Ledger Extension and other browser extensions?
Ledger Extension is the only browser extension built from the ground up to be compatible with a Ledger hardware wallet. With Ledger Extension, you can connect with dApps directly and seamlessly—without having to import your Ledger account into a software wallet.
Are other software wallets/extensions still compatible with Ledger wallets?

Launching Ledger Extension doesn’t mean that we are stopping or reducing collaboration with other wallets and browser extensions. With Ledger Extension, you simply have more freedom to choose how to connect Ledger wallets to your favorite dApps.

Troubleshooting

I'm having trouble connecting my Ledger Nano X to Ledger Extension via Bluetooth. How do I fix it?

In case of Bluetooth connection issues, follow these steps:

  1. Go to the Bluetooth settings of your desktop or smartphone.
  2. Remove your Ledger Nano X from the list of paired devices.
  3. Try connecting again.

If that doesn't work, try the following:

  1. Turn on and unlock your Ledger Nano X.
  2. Hold both buttons to access the Control Center.
  3. Navigate to Settings and press both buttons.
  4. Press the right button then press both buttons to access the Security menu.
  5. Press both buttons to validate Reset pairings.
  6. Confirm Reset pairings once more and enter your PIN code.
  7. Try connecting again.

Ledger Extension will still be functional to use, as a reminder the security of your experience comes from your private key being secured by the Ledger device that you use with it.
