How to fully uninstall Ledger Live

This tutorial describes how to uninstall Ledger Live Desktop on OS Windows, Mac, and Linux. 

 Windows  Mac  Linux

The easiest way to uninstall on Windows

  1. Click   Start on the taskbar and search for apps & features.
  2. Search for Ledger Live in the list of applications.
  3. Once the Ledger Live app is found, click on the app.
    The Uninstall button appears.
  4. Click Uninstall and confirm the choice.

  5. If the following warning appears, click Yes to allow the uninstallation of Ledger Live.

  6. Before starting the uninstallation, make sure to close the Ledger Live application.
  7. Click Uninstall and wait until the application is fully deleted from your PC.

    Ledger Live is now uninstalled. Let's go ahead and finish uninstalling by deleting the saved data directory.

How to delete saved data directory on Windows

  1. Press Windows Key and R (opens Run window).
  2. Type in %appdata% and Enter.
  3. Locate Ledger Live folder and right-click to Delete.
  4. Restart your PC.

How to fully uninstall on Mac OS

  1. Open the Macintosh HD folder located on the desktop.
  2. Click on the dropdown arrow beside Applications, then find the Ledger Live application and drag it to the Trash to delete it.
  3. Next, close the Applications dropdown menu by clicking on the arrow once more, then press Command + Shift + . (period) to unlock the hidden files.
  4. Click on the dropdown arrows beside Users > your.name > Library > Application Support, then delete the entire Ledger Live folder.
  5. Delete the entire Ledger Live folder by dragging it to the Trash.

How to uninstall on Linux 

  1. Navigate to the user data folder ~/.config/Ledger Live
  2. Delete all files in the Ledger Live folder.
  3. Delete the Ledger Live folder as well.

What's next?

After fully deleting Ledger Live from your computer, please reinstall it with a fresh version of Ledger Live located here.
