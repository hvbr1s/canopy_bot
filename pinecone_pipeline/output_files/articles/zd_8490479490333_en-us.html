How to access Ledger Live user data file

This article describes how to access your Ledger Live data file app.json stored on your computer. app.json contains information about your accounts, settings, and caches. This file can be very helpful if you want to transfer your data file to another PC or for troubleshooting certain issues as instructed by the Customer Support team.

Access app.json in Ledger Live desktop Access app.json on Windows Access app.json on macOS
To access the app.json file directory, you can use Ledger Live desktop. 
  1. Click Settings.
  2. Select the Help tab.
  3. Click on View next to View user data setting. A directory with an app.json file will open.
In some situations, you may not be able to access the app.json directory via Ledger Live. You can access it manually on Windows.
  1. Press the keys Windows + R at the same time. 
  2. In the appeared Run window, type %appdata% and press Enter.
    The roaming folder opens.
  3. Locate the Ledger Live folder and open it.
  4. Locate the app.json file.
In some situations, you may not be able to access the app.json directory via Ledger Live. You can access it manually on macOS.
  1. Open Finder.
  2. On the panel at the top select Go.
  3. Press and hold the Option button.
  4. Select appeared Library.

  5. Open the Application Support folder.
  6. Locate the Ledger Live folder and open it.
  7. In this folder, locate the app.json file.
