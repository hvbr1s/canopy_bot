Use Chainlist to connect MetaMask to any EVM network

This tutorial describes how to connect MetaMask to any Ethereum-compatible (EVM) network using Chainlist.

Before you start

  • Update Ledger Live to the latest version by using this link.
  • Connect your Ledger device to Ledger Live and check that it has the latest firmware version.
  • Make sure the latest version of the Ethereum (ETH) app is installed on your Ledger device.
  • Ensure your Ledger is connected to MetaMask. Learn more.

Step-by-step

  1. Unlock MetaMask and select your Ledger account.



  2. Go to https://chainlist.org/.
  3. Click Connect Wallet.



  4. Select your Ledger account then click Next.



  5. Select Connect.



  6. Select a network, click Add to MetaMask then click Approve.
  7. Select Switch network to confirm the connection.



    You have been successfully connected! You can now return to MetaMask and view the network and balance, which should have been updated.
