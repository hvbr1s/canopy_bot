API HTTP 503 error on Bitcoin account

This article explains how to solve API HTTP 503 errors on Bitcoin accounts.

API HTTP 503 error

This error may appear on sending a Bitcoin account in Ledger Live. It also can result in a synchronization error

In most cases, this error happens due to the outdated Ledger Live version and as a consequence, outdated firmware of the Ledger device and outdated BTC app on your Ledger device. 

Troubleshooting

Make sure to update Ledger Live to the latest version. 

In some cases, you may need to update Ledger Live manually by downloading it from the Ledger website. Updating manually is safe and all your existing accounts will remain if you install the new version over the old version. 

Find detailed instructions on updates below:

  • update Ledger Live
  • update Nano S / Nano X / Nano S Plus
  • update all apps on the Ledger device
