"T is not a function" Error message in Ledger Live

This support article provides information to users who encounter a "T is not a function, R is not a function" or "A is not a function" error message while using Ledger Live.

Before you start

  • Be sure to have the most updated version of Ledger Live. 
  • Update the firmware on your Ledger device Nano S / Nano S Plus / Nano X.

If you receive an error message saying "t is not a function," it's likely that the issue may be related to a proxy (work computer) or VPN causing interference with the connection to Ledger Live. To resolve the problem, we recommend trying the following steps:

  1. Switch to a different internet source: If you are using a Wi-Fi network, try connecting to a different network or use a mobile hotspot instead. Sometimes, certain networks can block or interfere with Ledger Live's connectivity.

  2. Disable proxy settings: If you are using a proxy server to connect to the internet, try disabling it and then launch Ledger Live. Proxy settings can sometimes interfere with the application's ability to communicate with the blockchain. In some cases, when using a work computer, a proxy may cause disruptions in communication with Ledger Live.

  3. Check your VPN. If you have a VPN enabled, activate Experimental networking in Ledger live by going into Settings > Experimental features. If the problem persists, turn off your VPN.

     

  4. Try using Ledger Live mobile instead. If you continue to encounter the "t is not a function" error message while using Ledger Live on your computer, you can try using the mobile app instead. The Ledger Live mobile app is available for both iOS and Android devices and can be downloaded from the respective app stores.

As always, if the issue still persists and you need assistance, always feel free to reach out to us so we can help resolve the issue.
