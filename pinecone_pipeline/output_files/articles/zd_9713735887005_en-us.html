Hiding 0-value transactions in Ledger Live

Starting with Ledger Live 2.54.0 and Ledger Live mobile 3.16.1, 0-value transactions will be automatically hidden in Ledger Live.

Hiding these transactions serves as protective measure against address poisoning scams.

Although it is possible to deactivate this protection by accessing Settings > Accounts and turning off the Hide token transaction when the value is 0 option, it is advisable to keep it enabled.
